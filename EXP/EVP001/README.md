# EVP001 で行った工夫（簡易メモ）

この実験では、既存の特徴量に加えて「天球座標の扱い」を強化し、分類精度の改善を狙いました。

## 1) 銀河座標への変換を追加
- `alpha`, `delta`（赤道座標）から `l`, `b`（銀河座標）を算出
- 星の分布構造（銀河面方向）を特徴量としてモデルに渡せるようにした

## 2) 角度特徴量の周期性を反映
- `alpha`, `delta`, `l`, `b` それぞれに対して `sin/cos` を追加
- 角度の 0°/360° 境界で不連続にならないようにした

## 3) 既存の有効特徴量は維持
- 色指数（`u_g`, `g_r`, `r_i`, `i_z`, `u_r`, `g_i`, `u_z`）を継続利用
- `redshift_log1p` も継続利用

## 4) 評価の進め方
- 3-fold CV の OOF で比較
- 重み付きアンサンブル + fold内バイアス補正を適用
- 追加でスタッキングも試行し、効果を確認

## 5) 結果サマリ
- 銀河座標 + 周期特徴量の追加で、ベース構成より OOF が改善
- 一方で `alpha`, `delta` を完全に除外すると悪化しやすく、共存が有効
